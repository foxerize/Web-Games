# Web Games

A static landing page for a browser-game collection.

## Publish on GitHub Pages

1. Push this repository to GitHub with `main` as its default branch.
2. Open **Settings → Pages** in the GitHub repository.
3. Under **Build and deployment**, set **Source** to **GitHub Actions**.
4. Push to `main` (or run **Deploy static site to GitHub Pages** from the Actions tab).

The deployment workflow publishes the repository root, so `index.html` is the site entry point.
